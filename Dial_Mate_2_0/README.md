# Dial Mate 2.0

A production-ready Urdu AI calling agent dashboard for Shopify stores with COD confirmation workflows, call analytics, billing, onboarding, and live order operations in a polished SaaS interface.

### Original Prompt

> A production-ready Urdu AI calling agent for Shopify that automatically calls customers after checkout to confirm COD/prepaid orders in natural, polite Urdu, detect intent, and answer questions about price, variants, delivery ETA, returns, exchange policy, stock, and order changes in real time from live Shopify data.

- Elegant SaaS dashboard with emerald, midnight, and ivory palette, clean analytics cards, call recordings, transcripts, order status timeline, and one-click “Call Now” / “Retry” actions.
- Human-like voice agent with Urdu-first conversation flow, Roman Urdu understanding, interruption handling, sentiment detection, fallback to human transfer, voicemail logic, and smart fraud/duplicate-order risk scoring.
- Deep Shopify integration: auto-sync products, FAQs, shipping zones, return policies, discount rules, and order edits; updates tags/statuses like Confirmed, No Answer, Cancelled, Reschedule.
- Revenue features: tiered pricing plans, usage-based billing, free trial, multilingual add-on, team seats, white-label option, and embeddable “AI Order Confirmation” widget on the storefront.
- Live deployment with onboarding wizard, secure cloud hosting, admin roles, webhook automation, compliance logs, and mobile-friendly interface.
